Ejemplo de como agregar código en la documentación de lib.rs con `````` dentro de `///`.

Generación de la documentación con `cargo doc --no-deps --open`

Si se ejecuta cargo test, ejecutara el código dentro de la documentación como tests

`cargo test`
running 2 tests
test src/lib.rs - overlap (line 10) ... ok
test src/lib.rs - overlap (line 4) ... ok

test result: ok. 2 passed; 0 failed; 0 ignored; 0 measured; 0 filtered out; finished in 0.00s
